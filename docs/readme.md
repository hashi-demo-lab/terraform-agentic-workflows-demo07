# Documentation

## Website

`docs/index.html` is a self-contained, single-file documentation site for the **HCP Terraform x AI Agents** project. It covers:

- **Foundations** — core concepts and architecture
- **Guardrails** — security controls and constraints
- **Spec-Driven Development** — the 4-phase SDD workflow
- **Workflows** — module, provider, and consumer authoring
- **Reference** — commands, templates, and configuration
- **Prerequisites** — tooling and environment setup

### Opening locally

Open the file directly in your browser — no build step or server required:

```bash
open docs/index.html
```

Or with a specific browser:

```bash
open -a "Google Chrome" docs/index.html
```

You can also navigate directly to the file URL:

```
file:///Users/<you>/git/terraform-agentic-workflows/docs/index.html
```
